P_cr = 2√(g_eff × EI)
EI = 6.34 MN·m²（60kg/m鋼軌）
g_eff = 橫向阻力 × 所有折扣係數
P_thermal = σ × A（A≈7.69×10⁻³m²）
▶ 完整公式推導
臨界屈曲力（UIC 720R §6）： P_cr = 2√(g · EI) g：有效橫向阻力（kN/m） EI：60kg軌斷面剛度 ≈ 6.34 MN·m² P_thermal：軌道承受之熱壓力 σ = E·α·ΔT（MPa） P_thermal = σ × A（A=7.69×10⁻³ m²） FS = P_cr / P_thermal FS > 1.5：充裕安全 1.0 ≤ FS < 1.5：警戒，應加強巡檢 FS < 1.0：高風險，建議降速或停駛

核心論點：現有CWR設計中和溫度（Neutral Temperature）採用歷史氣候假設。 隨著台灣升溫趨勢明確，設計時的最高軌溫上限可能已無法反映當前氣候條件， 熱應力安全餘裕正在被氣候變遷逐年侵蝕。建議設計階段採用近10年實測統計數據， 並每5年或重大維修後複查鎖定溫度設定值。
